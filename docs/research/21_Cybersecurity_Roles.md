# Cybersecurity Roles

## Purpose

This document defines the cybersecurity career paths supported by CyberPath AI.

Each role contains the required technical skills, recommended certifications, expected projects, and estimated learning progression.

---

# Supported Roles

## SOC Analyst

Description

Monitors security alerts, investigates incidents, and responds to threats.

Core Skills

- Networking
- Linux
- Windows
- SIEM
- Log Analysis
- Incident Response
- MITRE ATT&CK

Tools

- Splunk
- Wazuh
- Microsoft Sentinel

Projects

- SIEM Home Lab
- Log Analyzer
- Threat Detection Dashboard

Certifications

- Google Cybersecurity
- CompTIA Security+
- Blue Team Level 1

---

## Penetration Tester

Description

Identifies vulnerabilities by simulating attacks.

Core Skills

- Linux
- Networking
- Python
- Bash
- Nmap
- Burp Suite
- OWASP Top 10
- Active Directory

Projects

- Port Scanner
- Vulnerability Scanner
- Web Application Assessment

Certifications

- eJPT
- PNPT
- OSCP
