# User Flow

## Overview

The user journey is designed to be simple, guided, and beginner-friendly. Every step should move the learner closer to understanding their current cybersecurity readiness and what to learn next.

---

# Primary User Journey

Landing Page

↓

Select Target Career

↓

Choose Assessment Method

├── Upload Resume
└── Manual Skill Selection

↓

Set Weekly Study Hours

↓

Analyze Profile

↓

Dashboard

↓

Learning Roadmap

↓

Recommended Projects

↓

AI Mentor

---

# Landing Page

Purpose

Introduce the platform and allow users to start an assessment.

Actions

- Start Assessment
- Learn More

---

# Career Selection

Purpose

Allow users to select their target cybersecurity career.

Supported Careers

- SOC Analyst
- Penetration Tester
- Cloud Security Engineer
- Application Security Engineer
- Threat Intelligence Analyst
- Digital Forensics Analyst

---

# Assessment

Users may

- Upload Resume
- Select Existing Skills
- Specify Study Hours

---

# Analysis

Backend performs

- Resume Parsing
- Skill Extraction
- Skill Matching
- Gap Analysis
- Roadmap Generation

---

# Dashboard

Displays

- Career Readiness Score
- Current Skills
- Missing Skills
- Learning Timeline
- Recommended Projects

---

# Learning Roadmap

Displays

- Ordered Learning Path
- Estimated Duration
- Skill Dependencies
- Progress Tracking

---

# AI Mentor

Allows users to

- Ask Career Questions
- Understand Recommendations
- Receive Resume Suggestions

---

# End State

The learner leaves with

- Clear Career Goal
- Personalized Roadmap
- Practical Projects
- Actionable Next Steps
