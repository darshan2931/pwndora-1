# Product Requirements

# Functional Requirements

## User Profile

- Upload Resume
- Manual Skill Selection
- Choose Career Goal
- Set Weekly Learning Hours

---

## Career Intelligence

- Extract Existing Skills
- Compare Against Career Requirements
- Detect Missing Skills
- Calculate Career Readiness

---

## Learning Engine

- Generate Learning Roadmap
- Recommend Practical Projects
- Suggest Certifications
- Suggest Learning Resources

---

## AI Features

- Explain Recommendations
- Career Mentor Chat
- Learning Advice
- Resume Feedback

---

# Non-Functional Requirements

- Fast Response (<5 seconds)
- Simple User Experience
- Explainable AI
- Modular Architecture
- Easy Future Expansion

---

# User Stories

As a beginner,
I want to know where to start learning cybersecurity.

As a student,
I want a roadmap based on my current skills.

As a career switcher,
I want to understand what skills I need for my target role.

As a learner,
I want practical projects that improve my employability.

---

# Success Metrics

- Resume successfully analyzed
- Career readiness calculated
- Personalized roadmap generated
- User receives project recommendations
- AI provides contextual explanations
