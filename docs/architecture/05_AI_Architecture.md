# AI Architecture

## Overview

The AI layer is responsible for transforming structured cybersecurity data into personalized, human-readable guidance.

Unlike traditional AI assistants, CyberPath AI does not rely on the language model for decision making.

Business logic remains deterministic inside the backend.

The AI acts as an intelligent explanation and mentoring layer.

---

# Responsibilities

The AI Service is responsible for:

- Resume understanding
- Skill normalization
- Career explanations
- Learning recommendations
- AI Mentor Chat
- Resume improvement suggestions

## Resume Feedback

---

# Prompt Engineering Rules

- Always provide user context.
- Never send raw resumes directly.
- Request structured JSON whenever possible.
- Validate responses before sending them to the frontend.

---

# Safety

The AI must never:

- Invent certifications
- Invent skills
- Recommend unrelated technologies
- Change the learning order generated by the backend
