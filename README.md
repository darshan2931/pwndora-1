# CyberPath AI

## AI-Powered Cybersecurity Career Intelligence Platform

CyberPath AI helps learners understand where they are in their cybersecurity journey, identify missing skills, and generate personalized learning roadmaps using AI-assisted career intelligence.

---

## Features

- Resume Analysis
- Manual Skill Assessment
- Career Readiness Score
- Skill Gap Analysis
- Personalized Learning Roadmap
- AI Mentor
- Project Recommendations
