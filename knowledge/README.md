# CyberPath AI Knowledge Base

Structured cybersecurity knowledge used by the Career Intelligence Engine.

## Structure

```
roles/            # Cybersecurity role definitions
skills/           # Skill taxonomy with prerequisites
projects/         # Practical project catalog
certifications/   # Certification mappings
resources/        # Curated learning resources
```

Each file follows the schema defined in `docs/contracts/34_Knowledge_Base_Schema.md`.
